# Roadmap

## v1.0

- 125-document frozen benchmark
- 90 patients
- OCR, direct VLM, and hybrid tracks
- frozen statistics and publication assets
- community submission validation scaffold

## Next

- stronger community leaderboard workflow
- more explicit dataset onboarding helpers
- benchmark registry for reusable lane metadata
- richer submission templates and report generation

## Future

- larger benchmark releases
- multilingual coverage
- broader hospital diversity
- more document categories such as discharge summaries, pathology, and radiology-heavy bundles
- public benchmark versioning beyond the current frozen release
