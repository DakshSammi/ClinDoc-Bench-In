# Ground-Truth Template

This directory provides a public example of the canonical JSON annotation format used for benchmark ground truth.

Files included:

- `example_ground_truth.json`: example benchmark-style annotation in `CanonicalRawDoc` format

Use this when you want to:

- understand the expected annotation structure
- prepare your own dataset annotations
- compare benchmark-owned ground truth against model prediction JSON

For the full schema reference, see:

- [docs/schema.md](../../docs/schema.md)
- [docs/annotation_guide.md](../../docs/annotation_guide.md)
